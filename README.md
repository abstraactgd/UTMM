# UTMM Scripts & Data
This repository contains scripts and game data from Undertale Monster Mania (UTMM) and the official open-sourced UTMM Kit.

## Contents
**Data:** Weapons, Armor, SOULs, Battles, Food
**Scripts:**
- UTMM Kit: server-side and client-side source code
- Original UTMM: decompiled client-side attack pattern scripts (reference only)

## Purpose
Reference UTMM systems, hidden official content, and create or extend content using the UTMM Kit.
